Wetail No Duplicates

The plugin is meant to help in cleaning DB from any meta duplicates, such as:

  • User meta
  • Post meta
  • Term meta
  • Order item meta
  • Payment token meta
  • Comment meta

Before you launch any cleaning operation it is strongly advised to make a DB backup though all operations are totally fail-safe to run, because force-major cannot be predicted (as unexpected server shutdown).

Tables

In the list of tables it is recommended to clean only those which have more than 100 duplicates to not to run extra queries

First - check the number of duplicates for all tables or for those which are suspected to have ones.

Triggers

Triggers will prevent further occurrences of duplicates in selected tables. Absolutely safe to call.

Keep in mind that it is useful to check DB for duplicates from time to time anyways.
